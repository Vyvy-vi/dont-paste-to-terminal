Don't Copy - Paste Code!

Here's a brief demonstration of why you should not copy-paste in evaluation portals.

Go ahead and copy this console command.

console.log("I'm completely safe!");

Now paste it below:

Oh no! Here's what happened:

This little bit of JavaScript code detected your copy command and replaced your clipboard contents with something else. 

              
document.addEventListener("copy", (event) => {
    event.clipboardData.setData(
      "text/plain",
      `Hmmm this isn't right.`
    );
    event.preventDefault();
  });
